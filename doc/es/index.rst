===========================
Alerta dominios de internet
===========================

El módulo alerta dominios de internet envía un correo electrónico cuando expira
la renovación del dominio.

* Active en su compañía el envío de correos electrónicos cuando expire la
  renovación del dominio.
* Configure el planificador "Generar alertas de correos" para que se ejecute
  diariamente.
